About me

Hi, I’m Maike Anthony, a two-time Guinness World Memory Record holder [1] [2] . For over five years, I’ve taught memory and study techniques on my YouTube channel[3]. Along the way, I noticed a recurring problem: although the Memory Palace method is powerful, many people find it hard to apply consistently and get lost in the step-by-step process. Memory Palace Helper [4] was built to solve that problem. This app guides you through the Memory Palace method in a clear, systematic way, reducing friction and making the technique easier to use in real study sessions.

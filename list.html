  • How to memorize tables?

    If you want to memorize lists where each item has multiple attributes, like the countries or the elements, you can simply enter each item together with its features in the same text box separated by a space. E.g., "Hydrogen (H) 1", "Helium (He) 2", "Lithium (Li) 3", etc.

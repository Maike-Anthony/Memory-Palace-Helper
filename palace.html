Q&A

  • What does "locus" / "loci" mean?

    "Locus" (plural: loci) is the Latin word for "place" or "location." In the context of memory palaces, each locus is a position where you will imagine an image representing an item of the list.

  • How can I find memory palaces?

    You can use any place that you are familiar with as a memory palace. For example, you might use your home, workplace, or stores. You can find more ideas in the resources page.

  • How many loci can I place in one room?

    It's usually recommended to place around 5 loci per room, but this varies. The most important thing is to be able to distinguish a locus from one another. For example, if you are using a room with 5 indistinguishable chairs, it's probably better to only use one of them as a locus. Alternatively, you could use more than one of the chairs, but from a different angle.
